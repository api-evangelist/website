DrawLoop provides a SOAP and REST API for merging data together into a single documents, using a User ID and Password combination for authentication.

Using the DrawLoop LOOP API, developers can merge data from multiple links and other documents into a single PDF or other format, before sending, storing or printing.

Pricing for API access is available upon request, but they offer a FREE trial for new developers.

The Drawloop LOOP API for document merging is a perfect tool for the API stack.
